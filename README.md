# Nadeulhae (나들해)

> Weather intelligence + AI chat + vocabulary learning + collaborative code sharing

Full-stack Next.js 16 web application. Real-time collaborative code editor, FSRS spaced repetition, AI web-search chat, GPS-based weather scoring — all in one service.

## Quick Start

```bash
cd nadeulhae
cp .env.example .env.local    # edit with your credentials
npm install
npm run dev                    # http://localhost:3000
```

For production deployment, see [docs/ubuntu-server-deploy-auth.md](docs/ubuntu-server-deploy-auth.md).
